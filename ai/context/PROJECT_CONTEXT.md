# Project Context

AI Life OS helps an overwhelmed person capture anything in one place and recover only the context that matters.

## Current product focus

The long-term vision includes memory, goals, finance, Life Stats, insights, attention protection, and a companion. The current implementation focus is much smaller:

1. repository foundation;
2. one text capture;
3. durable raw-memory storage;
4. chronological history;
5. only then AI understanding.

## Canonical terminology

- **Memory** — immutable original user entry plus metadata.
- **Capture** — accepts and validates input, then asks Memory to persist it.
- **Understanding** — optional AI interpretation that creates proposals, never overwrites Memory.
- **Life Stats** — explainable snapshots of life domains; not a judgment of the person.
- **Attention Guardian** — future module that filters noise and detects overload.
- **Companion** — future conversational layer that advises but does not decide.

For conflict resolution, follow the single canonical rule in `PROJECT_BRAIN.md`.
