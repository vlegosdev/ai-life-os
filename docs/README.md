# Documentation

Markdown is the only canonical format in Git.

- `product/` — manifesto, vision, design principles, and the single roadmap.
- `engineering/` — architecture and engineering guidance.
- `adr/architecture-decisions.md` — the only durable decision journal.
- `operations/` — reference for AI handover; `AGENTS.md` is the executable rule source.

Documentation is frozen. Update it only for a code change, a discovered contradiction, or an accepted durable decision.
